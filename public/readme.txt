前端代码说明：
1、部署源码需安装nodejs，和express,path,http,multer,url,fs,body-parser等控件。并把项目放置到node服务器程序同级目录下，用node命令运行view-coder.js文件。
   Before we run viewcoder, we should have nodejs environment and install nmp, express, path, http,multer,url,fs,body-parser,etc. All those necessary node tools is writtern in view-coder.js file. Then we open terminal and run viewcoder.js with node command.

2、前端代码必须与后台相结合才能使用该平台，直接运行无法进行注册登录使用等。或直接访问http://viewcoder.beansonbar.cn网址进行查看效果
   Frontend code should operate with backend, and could not run individually if we try "login" or "sign in" function. Please visit "http://viewcoder.beansonbar.cn" and see how it operate
